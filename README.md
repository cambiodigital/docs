# Docs

This repository stores internal and public documentation used by the team.

## Purpose

- Keep documentation organized by product, service, or topic.
- Centralize reference material, guides, and technical notes.
- Make it easy to update and share documentation over time.

## Structure

- `kommo/`: documentation related to Kommo.

## Usage

- Add new documentation in a dedicated folder by topic.
- Prefer clear file names and simple Markdown structure.
- Update this README when new top-level sections are added.
